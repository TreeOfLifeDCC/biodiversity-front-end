Downloader Utility

Download data using a Python script. The downloader utility is a Python script used for downloading all assemblies, annotations, and experiments data. This utility can be run on any machine where Python or Docker is installed.

If you don't have Python installed, you can download it from the Python website. If you want to download Docker, you can get it from the Docker website.

There are two ways to run this utility:

1. Run the Python Script by Providing Parameters

Requirements:
  • Python installed
  • Run requirement.txt using the command: pip install -r requirement.txt
Run the script using the following command:

Enter the following command in the terminal:

python app/downloader-utility.py --project_name "DToL" --phylogeny "Chordata" --data_status "Mapped Reads - Done" --experiment_type "Chromium genome" --download_location "/Users/Documents" --download_option "assemblies" --species_list "Apamea sordens,Bufo bufo"

Note: The --phylogeny and --species_list parameters are conditional; you need to provide one of them.

2. Build and Run Docker Image

Requirements:
  • Docker installed and running
Create Docker build using the command:
docker build -t downloader-utility:latest .
Run Docker image by providing parameters:
docker run --rm -v "$PWD/download_location:/code/app/download_option" downloader-utility:latest --project_name "DToL" --phylogeny "Chordata" --experiment_type "Chromium genome" --download_option annotations --species_list "Apamea sordens,Bufo bufo"

Note: When using Docker, data is downloaded into the download_location (e.g., /Users/downloads), and download_option refers to either "annotations,assemblies" or "experiments".
